# 📚 Projeto Escola

Um site moderno e responsivo para uma instituição educacional, desenvolvido com HTML5, CSS3 e JavaScript puro.

## 🎯 Sobre o Projeto

Projeto Escola é uma plataforma educacional criada para compartilhar conhecimento de qualidade com foco em:
- **Educação de qualidade** com professores selecionados
- **Interface moderna e responsiva** para todos os dispositivos
- **Experiência do usuário otimizada** com animações suaves
- **Acessibilidade** garantida para todos os usuários

## 🗂️ Estrutura do Projeto

```
Projeto em equipe/
├── index.html                 # Página inicial
├── assets/
│   ├── css/
│   │   ├── style.css         # Estilos principais
│   │   ├── about.css         # Estilos da página "Sobre"
│   │   └── logim.css         # Estilos da página "Contato"
│   ├── js/
│   │   ├── index.js          # Scripts principais
│   │   └── contato.js        # Lógica do formulário de contato
│   ├── img/                  # Imagens do projeto
│   └── pages/
│       ├── about.html        # Página "Sobre-nós"
│       ├── logim.html        # Página "Fale Conosco"
│       └── Comfirmation/
│           └── comfirmation.html  # Página de confirmação
└── README.md                 # Este arquivo
```

## 🚀 Como Usar

### Abrir o projeto
1. Clone ou baixe este repositório
2. Abra `index.html` em um navegador web

### Visualizar as páginas
- **Página Inicial**: `index.html`
- **Sobre-nós**: `assets/pages/about.html`
- **Fale Conosco**: `assets/pages/logim.html`
- **Confirmação**: `assets/pages/Comfirmation/comfirmation.html`

## 💻 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Layouts modernos (Flexbox, Grid)
- **JavaScript (Vanilla)** - Interatividade e validação
- **LocalStorage** - Armazenamento de dados do formulário

## 📋 Funcionalidades

### Página Principal
- Hero section com chamada principal
- Apresentação do projeto
- Botão de contato destacado
- Footer com informações

### Página Sobre-nós
- Apresentação da equipe
- Perfis dos desenvolvedores
- Cards informativos com hover
- Design atrativo

### Página de Contato
- Formulário completo e validado
- Validação de e-mail
- Suporte a telefone e assunto
- Área de mensagem com limite de caracteres
- Envio para página de confirmação

### Página de Confirmação
- Resumo dos dados enviados
- Confirmação de sucesso

## 📱 Responsividade

O projeto é totalmente responsivo e se adapta para:
- ✅ Desktop (1400px+)
- ✅ Tablet (768px - 1400px)
- ✅ Mobile (até 768px)
- ✅ Small Mobile (até 480px)

## ♿ Acessibilidade

- ✅ Atributos `aria-label` nas seções
- ✅ Navegação por teclado suportada
- ✅ Contraste adequado entre cores
- ✅ Textos alternativos em imagens
- ✅ Semântica HTML5 apropriada

## 📊 Performance

- Scroll suave para uma experiência melhor
- Lazy loading de elementos com Intersection Observer
- CSS otimizado com media queries eficientes
- JavaScript sem dependências externas

## 🔒 Segurança

- Validação de formulário no cliente
- Armazenamento local seguro (localStorage)
- Nenhuma informação sensível exposta
- URLs relativas para melhor compatibilidade
